# Инструкция по работе с Google Sheets для новостей и подписчиков

## Как это работает

Сайт автоматически загружает данные из вашей Google Sheets таблицы при открытии страницы:
- **Новости** - из первого листа (gid=0)
- **Подписчики соцсетей** - из второго листа (gid=1)

Вам нужно только редактировать таблицу и нажимать кнопку "Обновить" на сайте.

## Настройка таблицы

### 1. Структура таблицы для НОВОСТЕЙ (Лист 1)

| photoUrl | title | description |
|----------|-------|-------------|
| https://example.com/image1.jpg | Заголовок новости 1 | Описание новости 1 |
| https://example.com/image2.jpg | Заголовок новости 2 | Описание новости 2 |

**Важно:**
- Первая строка - это заголовки колонок
- Названия колонок могут быть на русском: `фото`, `название`, `описание`
- Или на английском: `photoUrl`, `title`, `description`
- Регистр не важен (можно писать `Photo`, `PHOTO`, `photo`)

### 2. Структура таблицы для ПОДПИСЧИКОВ (Лист 2)

| platform | subscribers |
|----------|-------------|
| discord  | 1500        |
| telegram | 2300        |
| youtube  | 0           |
| twitch   | 850         |

**Важно:**
- Первая строка - это заголовки колонок
- Названия колонок могут быть на русском: `платформа`, `подписчики`
- Или на английском: `platform`, `subscribers`
- Названия платформ должны быть: `discord`, `telegram`, `youtube`, `twitch` (строчными буквами)
- Количество подписчиков - целое число

### 3. Публикация таблицы

1. Откройте вашу Google Sheets таблицу
2. Нажмите **Файл** → **Опубликовать в Интернете**
3. Выберите нужный лист:
   - Для новостей: Лист 1 (gid=0)
   - Для подписчиков: Лист 2 (gid=1)
4. Формат: **Веб-страница**
5. Нажмите **Опубликовать**
6. Скопируйте полученную ссылку

### 4. Добавление ссылок в код

#### Для новостей:
Откройте файл `src/components/sections/NewsSection.tsx` и замените ссылку:

```typescript
const SHEET_URL = 'ВАША_ССЫЛКА_ДЛЯ_НОВОСТЕЙ';
```

#### Для подписчиков:
Откройте файл `src/components/sections/SocialSection.tsx` и замените ссылку:

```typescript
const SUBSCRIBERS_SHEET_URL = 'ВАША_ССЫЛКА_ДЛЯ_ПОДПИСЧИКОВ';
```

Ссылки должны выглядеть примерно так:
```
Новости (gid=0):
https://docs.google.com/spreadsheets/d/e/2PACX-1vS0xXqmrU4t8oDUET_LDmpzOnCqA4kROUbl69bMS3LHhGfI-tIdBaX3LKY3VZUgWx4c6-BP_MOgdz-3/pub?gid=0&single=true&output=csv

Подписчики (gid=1):
https://docs.google.com/spreadsheets/d/e/2PACX-1vS0xXqmrU4t8oDUET_LDmpzOnCqA4kROUbl69bMS3LHhGfI-tIdBaX3LKY3VZUgWx4c6-BP_MOgdz-3/pub?gid=1&single=true&output=csv
```

**Важно:** В конце ссылки должно быть `&output=csv`

## Как добавлять новости

1. Откройте вашу Google Sheets таблицу (Лист 1)
2. Добавьте новую строку с данными:
   - **photoUrl**: Прямая ссылка на изображение
   - **title**: Заголовок новости
   - **description**: Описание новости
3. Сохраните изменения (Google Sheets сохраняет автоматически)
4. Откройте сайт и нажмите кнопку **"Обновить"** в разделе "Новости"

## Как обновлять подписчиков

1. Откройте вашу Google Sheets таблицу (Лист 2)
2. Измените количество подписчиков для нужной платформы
3. Сохраните изменения (Google Sheets сохраняет автоматически)
4. Откройте сайт и нажмите кнопку **"Обновить"** в разделе "Соц сети"

## Форматирование чисел подписчиков

Сайт автоматически форматирует большие числа:
- 1000 → 1.0K
- 15000 → 15.0K
- 1000000 → 1.0M
- 2500000 → 2.5M

## Требования к изображениям

- Используйте прямые ссылки на изображения (должны заканчиваться на `.jpg`, `.png`, `.webp` и т.д.)
- Рекомендуемый размер: 800x600 пикселей или больше
- Формат: JPG, PNG, WEBP
- Можно использовать сервисы: Imgur, Google Drive (с публичным доступом), или свой хостинг

## Устранение проблем

### Новости не загружаются

1. Проверьте, что таблица опубликована в Интернете
2. Убедитесь, что ссылка заканчивается на `&output=csv`
3. Проверьте названия колонок (должны быть `photoUrl`, `title`, `description`)
4. Убедитесь, что используется правильный лист (gid=0)
5. Откройте консоль браузера (F12) и посмотрите ошибки

### Подписчики не загружаются

1. Проверьте, что второй лист опубликован в Интернете
2. Убедитесь, что ссылка заканчивается на `&output=csv` и содержит `gid=1`
3. Проверьте названия колонок (должны быть `platform`, `subscribers`)
4. Проверьте названия платформ (должны быть строчными: discord, telegram, youtube, twitch)
5. Убедитесь, что количество подписчиков - это числа, а не текст

### Изображения не отображаются

1. Проверьте, что ссылки на изображения работают (откройте их в браузере)
2. Убедитесь, что изображения имеют публичный доступ
3. Используйте прямые ссылки на файлы изображений
