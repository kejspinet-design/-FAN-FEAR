# Инструкция по работе с Google Sheets для новостей, подписчиков и стафа

## Как это работает

Сайт автоматически загружает данные из вашей Google Sheets таблицы при открытии страницы:
- **Новости** - из колонок `photoUrl`, `title`, `description`
- **Подписчики соцсетей** - из колонок `platform`, `subscribers`
- **Стафф** - из колонок `nickname`, `role`, `avatar`

Все данные находятся на одном листе (gid=0). Вам нужно только редактировать таблицу и нажимать кнопку "Обновить" на сайте.

## Настройка таблицы

| photoUrl | title | description | platform | subscribers | nickname | role | avatar |
|----------|-------|-------------|----------|-------------|----------|------|--------|

| https://... | Новость 2 | Описание 2 | | | | | |
| | | | discord | 1500 | | | |
| | | | telegram | 2300 | | | |
| | | | youtube | 0 | | | |

| | | | | | Tiny | Основатель | |
| | | | | | Alt_Goty | Зам | |
| | | | | | Santa2555555 | Руководство | https://... |

**Важно:**
- Первая строка - это заголовки колонок
- Названия колонок могут быть на русском или английском
- Регистр не важен
- Можно оставлять пустые ячейки где данные не нужны

### Колонки для НОВОСТЕЙ

- **photoUrl** (или `фото`): Прямая ссылка на изображение
- **title** (или `название`, `заголовок`): Заголовок новости
- **description** (или `описание`): Описание новости

### Колонки для ПОДПИСЧИКОВ

- **platform** (или `платформа`, `соцсеть`): Название платформы (`discord`, `telegram`, `youtube`, `twitch`)
- **subscribers** (или `подписчики`, `количество`): Количество подписчиков (число)

### Колонки для СТАФА

- **nickname** (или `ник`, `никнейм`, `name`): Никнейм участника
- **role** (или `роль`, `должность`): Роль в команде
- **avatar** (или `аватар`, `фото`) - ОПЦИОНАЛЬНО: Ссылка на аватар

### Публикация таблицы

1. Откройте вашу Google Sheets таблицу
2. Нажмите **Файл** → **Опубликовать в Интернете**
3. Выберите нужный лист (обычно "Лист 1")
4. Формат: **Веб-страница**
5. Нажмите **Опубликовать**
6. Скопируйте полученную ссылку

### Добавление ссылки в код

Все три компонента используют одну и ту же таблицу (gid=0):

#### Для новостей:
Откройте файл `src/components/sections/NewsSection.tsx` и замените ссылку:

```typescript
const SHEET_URL = 'ВАША_ССЫЛКА';
```

#### Для подписчиков:
Откройте файл `src/components/sections/SocialSection.tsx` и замените ссылку:

```typescript
const SUBSCRIBERS_SHEET_URL = 'ВАША_ССЫЛКА';
```

#### Для стафа:
Откройте файл `src/components/sections/StaffSection.tsx` и замените ссылку:

```typescript
const STAFF_SHEET_URL = 'ВАША_ССЫЛКА';
```

Ссылка должна выглядеть примерно так:
```
https://docs.google.com/spreadsheets/d/e/2PACX-1vS0xXqmrU4t8oDUET_LDmpzOnCqA4kROUbl69bMS3LHhGfI-tIdBaX3LKY3VZUgWx4c6-BP_MOgdz-3/pub?gid=0&single=true&output=csv
```

**Важно:** В конце ссылки должно быть `&output=csv`

## Как добавлять данные

### Добавление новостей

1. Откройте вашу Google Sheets таблицу
2. Добавьте новую строку с данными в колонки `photoUrl`, `title`, `description`
3. Сохраните изменения (Google Sheets сохраняет автоматически)
4. Откройте сайт и нажмите кнопку **"Обновить"** в разделе "Новости"

### Обновление подписчиков

1. Откройте вашу Google Sheets таблицу
2. Измените количество подписчиков в колонках `platform`, `subscribers`
3. Сохраните изменения
4. Откройте сайт и нажмите кнопку **"Обновить"** в разделе "Соц сети"

### Добавление/изменение стафа

1. Откройте вашу Google Sheets таблицу
2. Добавьте или измените данные в колонках `nickname`, `role`, `avatar`
3. Сохраните изменения
4. Откройте сайт и нажмите кнопку **"Обновить"** в разделе "Стафф"

## Форматирование чисел подписчиков

Сайт автоматически форматирует большие числа:
- 1000 → 1.0K
- 15000 → 15.0K
- 1000000 → 1.0M
- 2500000 → 2.5M

## Требования к изображениям

- Используйте прямые ссылки на изображения (должны заканчиваться на `.jpg`, `.png`, `.webp` и т.д.)
- Рекомендуемый размер: 800x600 пикселей или больше
- Формат: JPG, PNG, WEBP
- Можно использовать сервисы: Imgur, Google Drive (с публичным доступом), или свой хостинг

## Устранение проблем

### Новости не загружаются

1. Проверьте, что таблица опубликована в Интернете
2. Убедитесь, что ссылка заканчивается на `&output=csv`
3. Проверьте названия колонок (должны быть `photoUrl`, `title`, `description`)
4. Убедитесь, что используется правильный лист (gid=0)
5. Откройте консоль браузера (F12) и посмотрите ошибки

### Подписчики не загружаются

1. Проверьте, что второй лист опубликован в Интернете
2. Убедитесь, что ссылка заканчивается на `&output=csv` и содержит `gid=1`
3. Проверьте названия колонок (должны быть `platform`, `subscribers`)
4. Проверьте названия платформ (должны быть строчными: discord, telegram, youtube, twitch)
5. Убедитесь, что количество подписчиков - это числа, а не текст

### Изображения не отображаются

1. Проверьте, что ссылки на изображения работают (откройте их в браузере)
2. Убедитесь, что изображения имеют публичный доступ
3. Используйте прямые ссылки на файлы изображений
